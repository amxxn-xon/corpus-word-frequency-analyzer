--- PROJECT OVERVIEW ---
This Python script is designed to ingest a raw text corpus, normalize and tokenize the string data, and mathematically sort the vocabulary to generate a Top 20 Word Frequency Leaderboard. 

--- FILES INCLUDED IN THIS ZIP ---
1. analyzer.py  -> The main Python script containing the execution logic.
2. corpus.txt   -> The raw input text (English Wikipedia summary on Tachyons).
3. results.txt  -> The exported analytics report generated by the script.
4. README.txt   -> This documentation file.

--- HOW TO EXECUTE ---
1. Extract the entire ZIP folder to your local machine.
2. IMPORTANT: Ensure that 'analyzer.py' and 'corpus.txt' remain in the EXACT same directory/folder.
3. Open the folder in your IDE (such as VS Code) or terminal.
4. Run 'analyzer.py'.
5. The terminal will print the Top 20 leaderboard, and 'results.txt' will automatically update with the fresh data.

--- NOTE ON METHODOLOGY ---
While advanced sorting methods (such as lambda functions from Lecture 9) are included as documented alternatives within the code, the primary engine was deliberately engineered using foundational control structures (Lists, Tuples, and Loops) to demonstrate a ground-up understanding of data manipulation.
